## Setup steps : 
### Git repository setup
1. Created a new folder for the project
    ```cmd
    C:\Users\Saksham> mkdir practice-resume-HTML
    C:\Users\Saksham> cd practice-resume-NTML
    ```
2. Initialized a Git repository in directory practice-resume-HTML
    ```cmd
    C:\Users\Saksham\practice-resume-HTML> git init
    ```
3. Created 2 new files - "README.md" (markdown readme file) and "resume.html" (main HTML file)
    ```cmd
    C:\Users\Saksham\practice-resume-HTML> touch README.md resume.html
    ```
4. Added all files to staging and commited with message "initial commit"
    ```cmd
    C:\Users\Saksham\practice-resume-HTML> git add .
    C:\Users\Saksham\practice-resume-HTML> git commit -m "initial commit"
    ```
### VS code setup
1. Opened the directory in VS code
    ```cmd
    C:\Users\Saksham\practice-resume-HTML> code . 
    ```
2. Created the resume web-page using pure semantic HTML with proper use of `<main>, <section>, <a> and <table>` tags.
